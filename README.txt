# ✈️ WanderWise — Travel Organizer App (WORK IN PROGRESS)

WanderWise is a full-stack travel organizer app that allows users to plan and visualize their entire trip in one place.

### 🚀 Features

- Add and store key trip details
- Visualize your trip with **pinpoints on an interactive map**
- Get **AI-powered destination suggestions** based on your travel preferences

---

## 🏗️ Tech Stack

### 🔧 Backend
- **Python**
- **Flask** – lightweight web framework
- **SQL** – for data storage (e.g., SQLite or PostgreSQL)
- **SQLAlchemy ORM** - for easy DB interaction)

### 💻 Frontend
- **React** – for dynamic UI
- **JavaScript**, **HTML**, **CSS** – for building and styling components
- **Map API** – (e.g., Google Maps, Mapbox) to display trip locations visually
